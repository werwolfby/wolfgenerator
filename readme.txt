
Gramma.atg

Gramma of WolfGenerator. It has been writen on CoCo/R. Use custom tool for generate compiler from this gramma.
For use this custom tool you should download CocoPlugin from CoCo Tools page (http://ssw.jku.at/Coco/Tools/VSPlugin)
Direct link on CoCo page: http://ssw.jku.at/Coco/Tools/VSPlugin/2010/Source.zip.

Plugin have some intellisense and colorize atg files.
